2.判断一个整数是否是回文数。回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。

示例 1:

输入: 121
输出: true
#include<stdio.h>
int main()
{
int i,n,j,m=0;
scanf("%d",&n);
j=n;
while(n>0){
i=n%10;
m=m*10+i;
n=n/10;}
if(m==j){
printf("true");}
else
printf("false");
return 0;
}
