判断一个整数是否是回文数。回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。


示例 1:


输入: 121

输出: true

示例?2:


输入: -121

输出: false

解释: 从左向右读, 为 -121 。 
从右向左读, 为 121- 。
因此它不是一个回文数。

示例 3:

输入: 10

输出: false

解释: 从右向左读, 为 01 。
因此它不是一个回文数。


bool isPalindrome(int x){
    int y,a;
    y=0;
    a=x;
    while(a!=0)
    {
        y*=10;
        y+=a%10;
        a/=10;
    }
    if(x>=y)return x==y;
    else return 0;
}
